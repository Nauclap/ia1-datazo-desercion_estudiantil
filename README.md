## Descripción del Proyecto

Este proyecto tiene como objetivo analizar y modelar el problema de la deserción y el éxito académico estudiantil mediante técnicas de exploración de datos, aprendizaje supervisado, aprendizaje no supervisado y reducción de dimensionalidad.

Se desarrollan modelos de inteligencia artificial para identificar patrones asociados al rendimiento académico de los estudiantes y predecir posibles escenarios de éxito, abandono o permanencia en sus estudios.

## Dataset Utilizado

### Nombre del Dataset

**Predict Students' Dropout and Academic Success**

### Descripción

Este conjunto de datos contiene información relacionada con características demográficas, antecedentes académicos, situación socioeconómica y desempeño estudiantil. Su propósito es facilitar el análisis y la predicción de la deserción académica y el éxito estudiantil.
